## ShoppingCarTest项目
* 该项目参照淘宝网购物车结算功能，支持用户勾选商品进行结算，提供结算后收货地址选择、订单确认等功能；
* Web前端为多页面项目，使用Vue.js框架搭建，借助vue-resource插件实现http数据请求。
